## иҫ№зјҳе®№еҷЁж—Ҙеҝ—иҫ“еҮәжҢҮеҜј<a name="ZH-CN_TOPIC_0000002479226424"></a>

**дҪҝз”ЁиғҢжҷҜ<a name="zh-cn_topic_0000001589264561_section15670165114555"></a>**

з”ұдәҺиҫ№зјҳи®ҫеӨҮпјҲеҰӮAtlas 500 A2 жҷәиғҪе°Ҹз«ҷпјүеӯҳеӮЁз©әй—ҙжңүйҷҗпјҢе№¶дё”иҫ№зјҳи®ҫеӨҮеӨҡйҮҮз”ЁeMMCзӯүflashдҪңдёәеӯҳеӮЁд»ӢиҙЁпјҢиҜҘд»ӢиҙЁеӯҳеңЁдҪҝз”ЁеҜҝе‘Ҫзҡ„йҷҗеҲ¶гҖӮдёәйҒҝе…ҚеӯҳеӮЁз©әй—ҙиҝҮеҝ«иў«еҶҷж»Ўд»ҺиҖҢеҪұе“ҚдёҡеҠЎжҲ–еӯҳеӮЁд»ӢиҙЁиҝҮеҝ«иҫҫеҲ°дҪҝз”ЁеҜҝе‘ҪпјҢз”ЁжҲ·еҸҜд»ҘеҸӮиҖғжң¬з« иҠӮиҫ№зјҳе®№еҷЁж—Ҙеҝ—зҡ„иҫ“еҮәе»әи®®пјҢдҪҝиҫ№зјҳе®№еҷЁд»ҘеҗҲйҖӮзҡ„ж–№ејҸиҫ“еҮәж—Ҙеҝ—гҖӮ

**иҫ“еҮәж–№ејҸ<a name="zh-cn_topic_0000001589264561_section5556162785617"></a>**

еҪ“еүҚAtlasзЎ¬д»¶дёҠиҝҗиЎҢзҡ„иҫ№зјҳе®№еҷЁеә”з”ЁдёҖиҲ¬жҳҜйҖҡиҝҮK8sе…је®№зҡ„иҫ№зјҳз®ЎзҗҶе№іеҸ°жқҘиҝӣиЎҢз®ЎзҗҶпјҢеҰӮеҚҺдёәдә‘IEFжҲ–еҹәдәҺKubeEdgeжҗӯе»әзҡ„з¬¬дёүж–№иҫ№зјҳе№іеҸ°зӯүгҖӮеңЁиҜҘе№іеҸ°дёӢпјҢе®№еҷЁж—Ҙеҝ—зҡ„иҫ“еҮәж–№ејҸдё»иҰҒеҲҶдёәд»ҘдёӢдёүз§Қпјҡ

- е®№еҷЁжҺ§еҲ¶еҸ°ж ҮеҮҶиҫ“еҮәпјҲSTDOUTе’ҢSTDERRпјүж–№ејҸ
- пјҲжҺЁиҚҗпјүжҢӮиҪҪеҲ°дё»жңәзӣ®еҪ•ж–№ејҸ
- е®№еҷЁж—Ҙеҝ—зӣҙжҺҘиҫ“еҮәеҲ°ж—Ҙеҝ—жңҚеҠЎ

>[!NOTE] 
>еҰӮжһңзі»з»ҹдёӯжңүж—Ҙеҝ—жңҚеҠЎеҷЁпјҢе»әи®®зӣҙжҺҘеңЁе®№еҷЁдёӯе°Ҷж—Ҙеҝ—иҫ“еҮәеҲ°ж—Ҙеҝ—жңҚеҠЎдёӯпјӣеҰӮжһңжІЎжңүпјҢе»әи®®йҮҮз”ЁжҢӮиҪҪеҲ°дё»жңәзӣ®еҪ•зҡ„ж–№ејҸиҫ“еҮәж—Ҙеҝ—пјҢеҮҸе°‘ж—Ҙеҝ—еҜ№зЎ¬д»¶е’Ңе…¶д»–дёҡеҠЎеҪұе“Қзҡ„йЈҺйҷ©гҖӮ

**е®№еҷЁжҺ§еҲ¶еҸ°ж ҮеҮҶиҫ“еҮәж–№ејҸ<a name="zh-cn_topic_0000001589264561_section8645749571"></a>**

еңЁиҝҷз§Қж–№ејҸдёӢпјҢеә”з”Ёе°Ҷе®№еҷЁзҡ„ж—Ҙеҝ—иҫ“еҮәеҲ°ж ҮеҮҶиҫ“еҮәгҖӮзјәзңҒжғ…еҶөдёӢпјҢDockerеј•ж“ҺжҚ•жҚүжүҖжңүе®№еҷЁзҡ„ж ҮеҮҶиҫ“еҮәпјҢдҪҝз”ЁJSONж јејҸеҶҷе…ҘеҲ°ж–Үд»¶йҮҢпјҢиҜҘж–Үд»¶дјҡдҝқеӯҳеҲ°дё»жңәзҡ„/var/lib/docker/containers/<i>\{containerid\}</i>зӣ®еҪ•дёӢпјҢеҰӮ[еӣҫ1](#zh-cn_topic_0000001589264561_zh-cn_topic_0000001182332559_zh-cn_topic_0000001092717454_fig167489420139)жүҖзӨәгҖӮ

>[!NOTE] 
>еҰӮжһңиҫ№зјҳз®ЎзҗҶе№іеҸ°дёҚж”ҜжҢҒиҜҘзӣ®еҪ•дёӢж—Ҙеҝ—ж–Үд»¶зҡ„з»•жҺҘжҲ–ж—Ҙеҝ—з»•жҺҘй…ҚзҪ®й”ҷиҜҜпјҢдјҡеҜјиҮҙ<b>/var/lib/docker</b>иў«еҚ ж»ЎпјҢд»ҺиҖҢеҪұе“Қж–°е®№еҷЁзҡ„йғЁзҪІеҸҠе…¶д»–е®№еҷЁдёҡеҠЎзҡ„жӯЈеёёиҝҗиЎҢгҖӮж•…дёҚе»әи®®йҮҮз”ЁиҜҘж–№ејҸгҖӮ

**пјҲжҺЁиҚҗпјүжҢӮиҪҪеҲ°дё»жңәзӣ®еҪ•ж–№ејҸ<a name="zh-cn_topic_0000001589264561_section139871046185718"></a>**

иҜҘж–№ејҸдёӢиҫ№зјҳе№іеҸ°ж—Ҙеҝ—ж”¶йӣҶзҡ„ж–№ејҸеҰӮ[еӣҫ2](#zh-cn_topic_0000001589264561_zh-cn_topic_0000001182452463_zh-cn_topic_0000001140102079_fig13294175199)жүҖзӨәгҖӮ

>- иҫ№зјҳе®№еҷЁз®ЎзҗҶе№іеҸ°дёҖиҲ¬дјҡж”ҜжҢҒиҜҘиғҪеҠӣпјҢд»ҘеҮҸе°‘еҜ№зі»з»ҹзӣ®еҪ•**var/lib/docker**зҡ„еҪұе“ҚгҖӮеҹәдәҺе®үе…ЁжҖ§иҖғиҷ‘пјҢиҜҘй…ҚзҪ®йңҖиҰҒз¬ҰеҗҲжүҖеңЁз»„з»Үзҡ„е®үе…ЁиҰҒжұӮгҖӮ

## й«ҳеҸҜз”ЁйӣҶзҫӨдёӯзҡ„и°ғеәҰз»„д»¶<a name="ZH-CN_TOPIC_0000002479226440"></a>

з”ҹдә§зҺҜеўғдёӯпјҢKubernetesйӣҶзҫӨйҖҡеёёдјҡйғЁзҪІеӨҡдёӘз®ЎзҗҶиҠӮзӮ№пјҢд»ҘйҒҝе…ҚеҚ•дёӘз®ЎзҗҶиҠӮзӮ№ж•…йҡңеҜјиҮҙж•ҙдёӘйӣҶзҫӨдёҚеҸҜз”ЁгҖӮKubernetesе®ҳж–№жҸҗдҫӣдәҶдёӨз§Қй«ҳеҸҜз”Ёзҡ„йӣҶзҫӨжҗӯе»әж–№жЎҲпјҢиҜ·еҸӮи§Ғ[Kubernetesж–ҮжЎЈ](https://kubernetes.io/zh/docs/setup/production-environment/tools/kubeadm/ha-topology/)дёӯй«ҳеҸҜз”ЁжӢ“жү‘йҖүйЎ№гҖӮйӣҶзҫӨи°ғеәҰз»„д»¶еҹәдәҺе®ҳж–№вҖң**Stacked etcd topology**вҖқж–№жЎҲиҝӣиЎҢйӘҢиҜҒпјҢеҗ„з»„д»¶иғҪеӨҹеңЁеӨҡдёӘз®ЎзҗҶиҠӮзӮ№еңәжҷҜдёӢжӯЈеёёиҝҗиЎҢпјҢдё”еҠҹиғҪжӯЈеёёгҖӮ

еӨҡз®ЎзҗҶиҠӮзӮ№еңәжҷҜйңҖиҰҒдҝқиҜҒжүҖжңүз®ЎзҗҶиҠӮзӮ№й…ҚзҪ®дёҖиҮҙпјҢеҰӮйӣҶзҫӨи°ғеәҰз»„д»¶й•ңеғҸгҖҒж—Ҙеҝ—зӣ®еҪ•гҖҒиҝҗиЎҢз”ЁжҲ·гҖҒиҠӮзӮ№ж Үзӯҫзӯүй…ҚзҪ®йңҖдёҖиҮҙгҖӮеӨҡз®ЎзҗҶиҠӮзӮ№дёӢйӣҶзҫӨи°ғеәҰз»„д»¶зҡ„е®үиЈ…иҜ·еҸӮи§Ғ[е®үиЈ…йғЁзҪІ](./installation_guide/03_installation.md)з« иҠӮгҖӮеӨҡз®ЎзҗҶиҠӮзӮ№зҡ„е®үиЈ…иҜ·еҸӮи§ҒK8sе®ҳж–№ж–ҮжЎЈ[еҲ©з”ЁkubeadmеҲӣе»әй«ҳеҸҜз”ЁйӣҶзҫӨ](https://kubernetes.io/zh-cn/docs/setup/production-environment/tools/kubeadm/high-availability/)гҖӮ

## дҪҝз”ЁContainerdзҡ„йӣҶзҫӨи°ғеәҰз»„д»¶<a name="ZH-CN_TOPIC_0000002479226400"></a>

Kubernetes  1.20зүҲжң¬д№ӢеҗҺпјҢе°ҶдёҚеҶҚж”ҜжҢҒдҪҝз”ЁDockerдҪңдёәCRIпјҲcontainer runtime interfaceпјүгҖӮз”ҹдә§зҺҜеўғдёӯпјҢеҰӮжһңеҜ№дҪҝз”Ёзҡ„K8sжңүй«ҳзүҲжң¬иҰҒжұӮж—¶пјҢйңҖиҰҒиҖғиҷ‘ж”№з”Ёе…¶е®ғCRIгҖӮйӣҶзҫӨи°ғеәҰз»„д»¶еҹәдәҺдё»жөҒCRI  Containerdзҡ„1.4.4зүҲжң¬иҝӣиЎҢе®үиЈ…е’ҢйӘҢиҜҒпјҢеҗ„з»„д»¶иғҪеӨҹеңЁContainerd  +  KubernetesеңәжҷҜдёӢжӯЈеёёиҝҗиЎҢпјҢдё”еҠҹиғҪжӯЈеёёгҖӮ

Containerdе®үиЈ…жөҒзЁӢиҜ·еҸӮи§Ғ[е®ҳж–№иө„ж–ҷ](https://github.com/containerd/containerd/blob/master/BUILDING.md)гҖӮйӣҶзҫӨи°ғеәҰз»„д»¶е®үиЈ…ж—¶й»ҳи®ӨдҪҝз”ЁAscend Docker RuntimeпјҢеҸҜд»Ҙй…ҚзҪ®ContainerdдҪҝз”ЁAscend Docker Runtimeжӣҝд»ЈruncпјҢз”ЁдәҺеңЁеҗҜеҠЁе®№еҷЁж—¶иҮӘеҠЁжҢӮиҪҪи®ҫеӨҮпјҢеҜ№ContainerdйңҖиҰҒеҒҡзҡ„й…ҚзҪ®иҜ·еҸӮи§Ғ[е®үиЈ…йғЁзҪІ](./installation_guide/03_installation.md)з« иҠӮзҡ„ContainerdеңәжҷҜдёӢе®үиЈ…Ascend Docker RuntimeгҖӮ

## иҮӘе®ҡд№үжҢҮж ҮејҖеҸ‘<a name="ZH-CN_TOPIC_0000002512192053"></a>

ж”ҜжҢҒйҖҡиҝҮеҰӮдёӢдёӨз§Қж–№ејҸејҖеҸ‘иҮӘе®ҡд№үжҢҮж ҮгҖӮ

- йҖҡиҝҮж–Үд»¶ж–№ејҸејҖеҸ‘иҮӘе®ҡд№үжҢҮж Ү

    з”ЁжҲ·ж №жҚ®[иҮӘе®ҡд№үжҢҮж Үж–Үд»¶](./api/npu_exporter.md#иҮӘе®ҡд№үжҢҮж Үж–Үд»¶)пјҢеҲӣе»әз¬ҰеҗҲиҰҒжұӮзҡ„иҮӘе®ҡд№үжҢҮж Үж–Үд»¶гҖӮеҗҜеҠЁNPU Exporterж—¶пјҢй…ҚзҪ®вҖң-textMetricsFilePathвҖқеҸӮж•°пјҢжҢҮе®ҡиҜҘиҮӘе®ҡд№үжҢҮж Үж–Үд»¶зҡ„и·Ҝеҫ„гҖӮиҜҰжғ…иҜ·еҸӮи§Ғ[NPU Exporter](./installation_guide/03_installation.md#npu-exporter)дёӯвҖңNPU ExporterеҗҜеҠЁеҸӮж•°вҖқиЎЁгҖӮNPU ExporterдјҡеңЁжҜҸдёӘж•°жҚ®йҮҮйӣҶе‘ЁжңҹиҜ»еҸ–иҮӘе®ҡд№үжҢҮж Үж–Үд»¶пјҢе№¶е°Ҷж–Үд»¶еҶ…е®№дёҠжҠҘз»ҷPrometheusжҲ–TelegrafгҖӮ

    ејҖеҸ‘зӨәдҫӢеҰӮдёӢпјҡ

    дҪҝз”ЁNPU ExporterйӣҶжҲҗе№¶йҮҮйӣҶDevkitе·Ҙе…·з”ҹжҲҗзҡ„hccs\_bandwidthжҢҮж ҮпјҢиҜҰжғ…иҜ·еҸӮи§Ғ[NPU ExporterйӣҶжҲҗDevkitйғЁзҪІжҢҮеҚ—](https://gitcode.com/Ascend/mindcluster-deploy/tree/master/samples/utils/npu-exporter)гҖӮе…ідәҺhccs\_bandwidthжҢҮж ҮдҝЎжҒҜзҡ„иҜҙжҳҺиҜ·еҸӮи§Ғ[HCCSеёҰе®Ҫзӣ‘жҺ§](https://www.hikunpeng.com/document/detail/zh/kunpengdevps/userguide/cliuserguide/KunpengDevKitCli_0251.html)гҖӮ

- йҖҡиҝҮжҸ’д»¶ж–№ејҸејҖеҸ‘иҮӘе®ҡд№үжҢҮж Ү

    з”ЁжҲ·еҸҜйҖҡиҝҮзј–еҶҷжҸ’д»¶зҡ„ж–№ејҸиҮӘе®ҡд№үжҢҮж ҮпјҢдҪҝз”ЁиҜҘжҸ’д»¶еүҚпјҢејҖеҸ‘иҖ…йңҖиҰҒиҮӘиЎҢеӯҰд№ дәҶи§ЈcgoгҖҒgoзӣёе…іиҜӯиЁҖзү№жҖ§пјҢе№¶йҳ…иҜ»[README](https://gitcode.com/Ascend/mind-cluster/blob/master/component/npu-exporter/plugins/README.md)дәҶи§ЈдҪҝз”Ёж–№жі•гҖӮ
